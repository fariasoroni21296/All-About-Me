On Going Research

Hand Gesture Recognition based Applications For Educational Purpose

This project on Hand gesture recognition based applications for educational purpose is on improving the recognition of the human hand postures in online education system because of this Covid-19 pandemic. We are trying to develop an application for computer mouse control, sign gestures and a virtual blackboard using image processing.

The system uses nothing but a low-resolution webcam and it will be able to track the userâ€™s hand in two dimensions and can recognize trained hand gestures, which are defined as mouse functions, alphabets and numbers recognizing functions etc.
